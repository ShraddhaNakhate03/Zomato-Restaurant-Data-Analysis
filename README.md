# 🍽️ Zomato Restaurant Data Analysis

## 📊 Project Overview
This project performs **Exploratory Data Analysis (EDA)** on the Zomato restaurant dataset to uncover insights about restaurant trends, cuisines, pricing, and customer ratings.

The objective is to identify patterns that influence restaurant popularity and help businesses understand customer preferences.

---

## 📁 Dataset Information

Dataset: Zomato Restaurant Dataset  
Total Records: 9551 Restaurants  
Total Features: 20 Columns

Important attributes include:

- Restaurant Name
- City
- Cuisines
- Average Cost for Two
- Aggregate Rating
- Votes
- Price Range
- Online Delivery
- Table Booking
- Latitude & Longitude

---

## 🎯 Business Questions Solved

This analysis answers important business questions such as:

- Which **cities have the highest number of restaurants?**
- What are the **most popular cuisines?**
- Does **higher price mean better ratings?**
- Do restaurants with **online delivery get more votes?**
- Which restaurants have the **highest ratings?**
